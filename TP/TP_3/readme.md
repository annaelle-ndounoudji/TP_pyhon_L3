# Gestion des Étudiants

## Description
Programme qui gère une liste d'étudiants avec leurs notes, affiche les admis et calcule la moyenne de la classe.

## Comment l'utiliser
```bash
python etudiants.py
```

## Fonctionnalités
- Affiche les étudiants admis (moyenne ≥ 10)
- Calcule la moyenne générale de la classe

## Exemple de résultat
```
Étudiants admis :
Alice
Clara
Moyenne générale de la classe : 11.666666666666666
```

## Structure des données
Chaque étudiant contient :
- `nom` : nom de l'étudiant
- `age` : âge de l'étudiant
- `moyenne` : moyenne sur 20
