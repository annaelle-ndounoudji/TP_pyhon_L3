# Analyseur de Phrase

## Description
Programme qui analyse une phrase saisie par l'utilisateur : compte les mots, trouve le mot le plus long et vérifie si c'est un palindrome.

## Comment l'utiliser
```bash
python analyse_phrase.py
```

## Fonctionnalités
- Compte le nombre de mots
- Trouve le mot le plus long
- Vérifie si la phrase est un palindrome

## Exemple d'utilisation
```
Entrez une phrase : La mariee ira mal
Nombre de mots : 4
Mot le plus long : mariee
La phrase est un palindrome
```
```
Entrez une phrase : Bonjour tout le monde
Nombre de mots : 4
Mot le plus long : bonjour
La phrase n'est pas un palindrome
```

## Qu'est-ce qu'un palindrome ?
Un palindrome est une phrase qui se lit de la même façon dans les deux sens (sans tenir compte des espaces).

Exemples : "elu par cette crapule", "esope reste ici et se repose"
