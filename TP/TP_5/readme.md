# Calculateur de Moyenne à partir d'un Fichier

## Description
Programme qui lit des notes depuis un fichier texte, calcule la moyenne et enregistre le résultat dans un autre fichier.

## Comment l'utiliser

### 1. Créer le fichier notes.txt
Crée un fichier `notes.txt` avec une note par ligne :
```
10
12
14
9
15
```

### 2. Exécuter le programme
```bash
python calcul_notes.py
```

### 3. Résultat
Le programme crée automatiquement `resultat.txt` contenant :
```
Moyenne de la classe : 12.0
```

## Fichiers utilisés
- **notes.txt** : fichier d'entrée contenant les notes (une par ligne)
- **resultat.txt** : fichier de sortie contenant la moyenne calculée

## Fonctionnalités
- Lecture automatique de toutes les notes
- Calcul de la moyenne
- Enregistrement du résultat dans un fichier
