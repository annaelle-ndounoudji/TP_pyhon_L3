# Gestion d'Étudiants

## Description
Application Python de gestion d'étudiants avec POO, sauvegarde JSON et statistiques.

## Fonctionnalités
- Ajouter des étudiants
- Enregistrer des notes
- Calculer moyennes et mentions
- Afficher statistiques
- Sauvegarde automatique

## Utilisation
```bash
python gestion.py
```

## Menu
1. Ajouter étudiant
2. Ajouter note
3. Rechercher
4. Afficher tous
5. Statistiques
6. Quitter

## Mentions
- < 10 : Ajourné
- 10-12 : Passable
- 12-14 : Assez bien
- 14-16 : Bien
- ≥ 16 : Très bien

## Fichiers
- `gestion.py` : programme principal
- `etudiants.json` : base de données

## Compétences
- POO (classes)
- Fichiers JSON
- Statistiques
- Menu interactif
